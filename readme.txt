== Description ==

This plugin adds a dynamic widget to your sidebar. You can set different widgets to show for different posts/pages, without the need to change the sidebar code.


== Installation for Wordpress 2.3.3 and below ==

1. Upload the folder 'wp-widget-changer' to the '/wp-content/plugins/' directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Presentation -> Widgets. Add the 'Widget Changer' widget to the sidebar and place it in the order that you want it to appear. Make sure that the 'Recent posts' widget is not placed above the 'Widget Changer' widget.
4. Go to Option->Widget Changer. Set the option for the user right and where the widget is to be show.
5. If you choose Post, then you can see the Widget Changer input field below the 'Publish' button in the Write section. 
6. If you choose Category, you can manage the widget from the Option->Widget Changer page.

== Installation for Wordpress 2.5 ==

1. Upload the folder 'wp-widget-changer' to the '/wp-content/plugins/' directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Design -> Widgets. Add the 'Widget Changer' widget to the sidebar and place it in the order that you want it to appear. Make sure that the 'Recent posts' widget is not placed above the 'Widget Changer' widget.
4. Go to Settings->Widget Changer. Set the option for the user right and where the widget is to be show.
5. If you choose Post, then you can see the Widget Changer input field below the text field in the Write section. 
6. If you choose Category, you can manage the widget from the Settings->Widget Changer page.

-- Upgrading --
1. First deactivate the previous version of the plugin at 'Plugins' menu in WordPress
2. Upload the latest version
3. Activate the plugin through the 'Plugins' menu
